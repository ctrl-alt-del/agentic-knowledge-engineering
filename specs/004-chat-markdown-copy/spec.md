# Chat Markdown + Copy — 规范

## 用户故事
作为用户，我希望聊天内容以格式化 Markdown 渲染，能够一键复制消息，
并能看到每条消息的发送时间。

## 功能需求

### Markdown 渲染
- [ ] `**粗体**` 渲染为 `<strong>粗体</strong>`
- [ ] `*斜体*` 渲染为 `<em>斜体</em>`
- [ ] `` `行内代码` `` 渲染为 `<code>行内代码</code>`
- [ ] ```` ```代码块``` ```` 渲染为 `<pre><code>代码块</code></pre>`
- [ ] `[链接](url)` 渲染为 `<a href="url">链接</a>`
- [ ] `- 项目` / `* 项目` 渲染为无序列表
- [ ] `1. 项目` 渲染为有序列表
- [ ] 空行分隔的文本渲染为段落 `<p>`
- [ ] 输入中的 HTML 标签被转义（防 XSS）

### 复制按钮
- [ ] 每条消息气泡下方右侧显示复制按钮（剪贴板图标）
- [ ] 点击后复制消息纯文本内容到剪贴板
- [ ] 复制成功后按钮文字短暂变为 "Copied!"（1.5 秒）
- [ ] Desktop：hover 时显示按钮；Touch：始终显示

### 时间戳
- [ ] 每条消息气泡下方左侧显示时间戳
- [ ] 格式：`YYYY-mm-dd HH:MM:SS`
- [ ] 字号 10px，颜色 muted

## 适用范围
- [ ] Vue 应用（`ui/`）的所有消息气泡
- [ ] Demo（`ui_lite/index.html`）的所有消息气泡

## 验收标准

### 正常流程
- [ ] 给定消息文本包含 `**bold**`，当渲染时，则显示为粗体文字
- [ ] 给定消息文本包含 `*italic*`，当渲染时，则显示为斜体文字
- [ ] 给定消息文本包含 `` `code` ``，当渲染时，则显示为等宽字体代码
- [ ] 给定消息文本包含 `- item`，当渲染时，则显示为列表项
- [ ] 给定消息文本包含 `<script>alert(1)</script>`，当渲染时，则以纯文本显示（不执行脚本）
- [ ] 给定用户点击复制按钮，当操作完成后，则剪贴板包含消息纯文本
- [ ] 给定新消息生成，当渲染时，则显示当前时间戳

### 边界情况
- [ ] 给定空消息文本，当渲染时，则不显示内容但时间戳和复制按钮仍存在
- [ ] 给定极长消息（5000+ 字符），当渲染时，则不出错、不截断
- [ ] 给定嵌套标记（如 `**bold *italic* bold**`），当渲染时，则正确渲染
- [ ] 给定 clipboard API 不可用，当点击复制时，则静默失败不报错

## 非功能需求
- Markdown 解析器无外部依赖（npm 或 CDN）
- 解析器可被 Vue 应用和 Demo 共用
- 首次渲染不引起明显抖动（<10ms per message）
- XSS 安全：用户输入中的 HTML 不执行
